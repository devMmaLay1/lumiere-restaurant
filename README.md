# Lumière - Fine Dining Restaurant Website

A sophisticated, fully responsive restaurant website template featuring elegant animations, smooth scrolling, and modern web design principles.

## 🌟 Features

- **7 Complete Pages**: Home, Menu, Reservations, About, Gallery, Events, Contact
- **Responsive Design**: Mobile-first approach, works perfectly on all devices
- **Smooth Animations**: GSAP animations and Lenis smooth scrolling
- **Dynamic Menu System**: Filter by category with pagination
- **Interactive Gallery**: Filterable image gallery with lightbox
- **Reservation System**: Beautiful booking form with date/time selection
- **Component-Based**: Reusable navbar and footer components
- **Performance Optimized**: Lazy loading images, efficient animations

## 🚀 Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
- Lenis Smooth Scroll
- AOS (Animate On Scroll)
- CountUp.js

## 🎨 Customization

All customizable elements are marked with 🔧 EDIT comments in the code:

- **Colors**: Edit CSS variables in `assets/css/main.css`
- **Fonts**: Update Google Fonts links and CSS variables
- **Content**: All text content is in HTML files
- **Images**: Replace images in `assets/images/` folders

## 📱 Pages

1. **Home** - Hero section, features, testimonials
2. **Menu** - Filterable menu with pagination
3. **Reservations** - Booking form with date/time picker
4. **About** - Restaurant story, team, values
5. **Gallery** - Filterable image gallery with lightbox
6. **Events** - Special events and private dining
7. **Contact** - Contact form, map, info

## 🛠️ Setup

1. Clone the repository
2. Open `lumiere/index.html` in your browser
3. No build process required - pure HTML/CSS/JS
