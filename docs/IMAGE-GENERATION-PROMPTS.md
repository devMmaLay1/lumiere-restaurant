## GENERATION TIPS

**For Best Results:**

1. **Consistency:** Use the same AI model for all images to maintain visual consistency
2. **Aspect Ratio:** Generate at 3:2 ratio (e.g., 1200x800px) then crop to 600x400px
3. **Post-Processing:** Adjust brightness, contrast, and color temperature for consistency
4. **Background:** Keep backgrounds dark (black or dark slate) for luxury feel
5. **Lighting:** Soft overhead lighting with subtle shadows
6. **Plating:** Modern minimalist plating on dark or white plates
7. **Garnish:** Include microgreens, edible flowers, and fresh herbs
8. **Format:** Convert final images to WebP format for web optimization

**Recommended AI Tools:**
- Midjourney (best for food photography realism)
- DALL-E 3 (good for consistent style)
- Stable Diffusion XL (with food photography models)
- Adobe Firefly (commercial use friendly)

**Midjourney Parameters to Add:**
```
--ar 3:2 --style raw --stylize 250 --v 6
```

**DALL-E 3 Style Modifier:**
```
Style: photorealistic, professional food photography
```
